# 01 · Brainstorm — 영어 빙고

> brainstorming/SKILL.md 적용. 원격 무인 세션이라 AI 자체 판단.

## MUST (꼭 필요한 기능, 3개+)

1. **단어 풀 관리**
   - 텍스트 영역에 영단어 줄바꿈 입력 (또는 쉼표/공백 구분)
   - 최소 16개(4x4) / 25개(5x5) 검증
   - localStorage 영속 저장
   - 기본 샘플 단어 25개 내장 (3~4학년 수준 명사·동사)

2. **빙고판 생성**
   - 보드 크기 선택 4x4 / 5x5 토글
   - 학생 모드: 입력한 단어 풀에서 무작위로 칸 채움 (Fisher-Yates shuffle)
   - "다른 판" 버튼으로 재추첨 (인쇄 전 모둠별 다른 판 가능)
   - **인쇄 모드**: A4에 빙고판 1~4개 격자 배치

3. **교사 호명 모드**
   - 단어 풀 전체에서 무작위 1개 호명 (중복 없이)
   - 한글 뜻 큰 글씨 표시(영어 단어 옆 또는 토글로 숨김) — 학생이 들리지 않을 때 보조
   - TTS 발음 (Web Speech API, en-US)
   - 호명된 단어 자동 체크리스트 (남은 단어 수 표시)
   - "다시 듣기" / "다음 단어" 버튼

4. **자동 BINGO 감지 (학생 모드)**
   - 학생이 자기 칸 클릭 → 토글 마크
   - 가로/세로/대각선 라인 완성 시 빙고 효과 (색상 강조)
   - 라인 수(빙고 수) 카운트 표시

## SHOULD (있으면 좋은 기능, 2개+)

1. **속도·발음 조절**
   - TTS rate slider (0.6 ~ 1.2)
   - voice 선택 가능 (en-US 여성/남성)

2. **히스토리 패널**
   - 호명된 단어 목록 (영어 + 한글) 토글로 펼치기
   - "처음으로" 버튼으로 게임 리셋

3. **풀 화면 토글**
   - 교실 TV 띄울 때 메뉴 없는 큰 글씨 모드

## MUST NOT (절대 금지, 2개+)

1. **개인정보 수집 일체 금지**: 학생 이름·번호·점수·결과 입력란 자체를 만들지 않는다 (scope creep 방지). 빙고 우승자 기록도 없음.
2. **서버/네트워크 통신 금지**: 모든 로직 클라이언트 측, fetch/XHR 0건. (Gemini 비사용)
3. **사진/카메라 접근 금지**: getUserMedia 호출 없음.
4. **외부 CDN 의존 금지**: 헤드리스/오프라인 환경 호환. 모든 코드·스타일 인라인 또는 same-origin.

## 인터랙션 패턴
- 메인 페이지 = 1페이지 SPA
- 좌측: 단어 풀 입력·설정 (collapsible)
- 우측: 보드 (학생 미리보기) + 호명 패널
- 인쇄 시: settings 숨김, 보드만 출력

## 성공 기준
- 단어 25개 넣고 5x5 보드 생성, 인쇄 미리보기 작동
- 호명 모드에서 5개 단어 연속 호명 후 "다시 듣기" 정상
- 학생 칸 클릭 → 한 줄 완성 시 BINGO 시각 강조
- 새로고침 후 단어 풀 보존
